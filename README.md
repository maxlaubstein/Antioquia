# Antioquia Conservation Genomics
Scripts for assembling reference genomes for 2 endangered bird species in Antioquia, Colombia: Antioquia Brushfinch / Montañerito Paisa (Atlapetes blancae) and 'Paisa' Antpitta (Grallaria sp. nov.)
